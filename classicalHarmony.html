Classical Harmony

Personal Project

100%
MACLURE NETWORK

Classical Harmony displays key signatures and chords used in classical music composition in any key based on the user's selected key signature. Built with HTML, CSS, and Javascript

Based on the key signature a user slelects, Classical Harmony displays the following information for that key:

  • Key Signature
  • Diatonic Chords
  • Secondary Dominants (V and vii)
  • Subdominant Substitutions
  • Chromatic Mediants
  • Related Key Signatures
